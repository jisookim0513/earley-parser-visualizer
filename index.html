Grammars

  • This program only supports three identifiers
    • grammar: grammar E -> V
    • input: input 2+3+4
    • assoc: assoc left +
  • You should always put a space between identifier and context
    • e.g. grammar E -> V   ✓
    • e.g. grammarE -> V    ✗
  • You should always put a space between the arrow and
    each side of grammar
    • e.g. grammar E -> V   ✓
    • e.g. grammar E->V     ✗
  • You can use %prec and %dprec as following
    (don't forget space!):
    • e.g. grammar E -> V %dprec 1   ✓
    • e.g. grammar E -> V %dprec1    ✗
    • e.g. grammar E -> V%dprec 1    ✗
    • e.g. grammar E -> V %prec !   ✓
    • e.g. grammar E -> V %prec '!'    ✗
  • For assoc, you can have multiple procedures for the same tier:
    • e.g. assoc left +,!,-       ✓
    • e.g. assoc right +, !, -   ✓
    • e.g. assoc left +!-         ✗
